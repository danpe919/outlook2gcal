# auto-scheduler

## Dependencies

- Python 3.7+
- [Poetry](https://github.com/python-poetry/poetry)

## Installation

```bash
poetry install
```

## Preparation

1. [Enable Google Calendar API](https://developers.google.com/calendar/quickstart/python)
2. Download ``credentials.json`` to ``./creds``

## Run

### Linux

```bash
poetry run .src//main.py
```

### Windows

Run ``start.bat``.
